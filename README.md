# 🏡 Paradiso

A full-stack real estate platform that simplifies property buying and renting. Paradiso offers intuitive property search, listings management, and a seamless experience for both buyers and renters.

---

## ✨ Features

- 🔍 **Property Search** — filter listings by location, price, type, and more
- 🏠 **Listings Management** — add, edit, and remove property listings
- ❤️ **Save & Shortlist** — bookmark properties for later
- 📱 **Responsive Design** — smooth experience across mobile, tablet, and desktop
- 🔐 **User Authentication** — secure sign up, login, and session management

## 🚀 Getting Started

### Prerequisites
- Node.js v18+
- npm or yarn

### Installation

```bash
# Clone the repo
git clone https://github.com/ofeoritse-amiteye/paradiso.git
cd paradiso

# Install backend dependencies
cd server
npm install

# Install frontend dependencies
cd ../client
npm install
```

### Environment Variables

Create a `.env` file in the `/server` directory:

```env
PORT=5000
JWT_SECRET=your_jwt_secret
DATABASE_URL=your_database_url
```

### Running the App

```bash
# Start the backend
cd server
npm run dev

# Start the frontend (in a new terminal)
cd client
npm run dev
```
